## O que é o Binary Search
<a id="chapter-1"></a>
O Binary search é um algoritmo que faz uma **busca eficiente** para achar um item qualquer em uma ***lista ordenada***. A forma que funciona é muito simples: ele vai dividir a lista ao meio ( na parte que pode conter o elemento buscado ), até reduzir o suficiente para achar o elemento buscado, caso exista.
